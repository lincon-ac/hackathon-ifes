Capacite-se em programação web com o Projeto Reprograme-se

Aprenda habilidades práticas em programação web com o Projeto Reprograme-se, uma iniciativa que oferece capacitação para qualquer pessoa interessada. Inscreva-se agora e comece a construir sua carreira na área de tecnologia.

O Reprograme-se é um projeto que conta com toda experiência e competência do Ifes, uma instituição reconhecidamente de excelência na formação profissional, com a participação da Associação Capixaba de Tecnologia – ACT!ON.

Perguntas frequentes

O reprograme-se é composto por dois cursos: o curso Programação básica de aplicativos Web - Front-end e o curso Programação com JavaScript e o Framework Node.js - Backend (confira o link da estrutura do curso).
O reprograme-se terá a duração de 8 meses, sendo 4 meses para o curso Programação básica de aplicativos Web - Front-end e 4 meses para o curso Programação com JavaScript e o Framework Node.js - Back-end.
Não, o curso é totalmente gratuito.

Informações

Os cursos serão desenvolvidos pelo Ifes campus Cachoeiro de Itapemirim utilizando-se de Ambientes Virtuais de Aprendizagem e o aluno contará com orientações de profissionais capacitados como professores formadores, professores mediadores e o apoio de monitores.

Os estudos ocorrerão de forma online e, ainda, o aluno contará com o apoio e orientação de forma presencial no polo de apoio presencial de seu município.

sobre o IFES

O Ifes é uma instituição de referência em educação na sociedade capixaba, que promove educação profissional pública de excelência, integrando ensino, pesquisa e extensão. Foi criado a partir da união das unidades do Centro Federal de Educação Tecnológica e das Escolas Agrotécnicas Federais em 2008, com o objetivo de construir uma sociedade democrática, justa e sustentável.
